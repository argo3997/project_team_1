# 🏙️ 라이징 상권 발굴 프로젝트

> MZ세대가 선호하는 "제2의 성수동"을 데이터로 발굴하는 프로젝트

---

## 📌 프로젝트 개요

### 주제 선정 이유

홍대, 연남, 성수 등 핫플레이스를 다니면서 **"다음에 어떤 상권이 뜰까?"**라는 궁금증에서 시작했다.

현재 이 지역들은:

상권 과밀화로 웨이팅 증가, 소비자 경험 저하,
한류 영향으로 외국인 관광객까지 몰려 더욱 붐빔, 임대료 상승 + 빅 프랜차이즈 진입으로 자영업자들이 새로운 터전으로 이주 중

이러한 자영업자들의 이동이 상권 변화의 초기 신호가 된다고 판단했다.

### 목표

본 프로젝트는 성수동과 유사한 특성을 가지면서 **아직 MZ세대에게 포화되지 않은 "Post 성수동"**을 데이터 기반으로 발굴하는 것을 목표로 한다.

### 분석 과제

- 서울 전체 데이터만으로 분석하면 "성수동"과의 연결고리가 약함
- 성수동 데이터만으로 학습하면 샘플 수 부족(104개)으로 통계적 유의성 확보 불가
- **해결책**: 두 접근법의 장점을 결합한 **하이브리드 모델** 개발

---

## 🔬 분석 방법론

### 하이브리드 모델 구조

```
하이브리드 점수 = Model 2 점수 × 0.7 + 성수 유사도 점수 × 0.3
```

| 모델        | 데이터               | Prob(F)    | 역할               | 가중치  |
| ----------- | -------------------- | ---------- | ------------------ | ------- |
| Model 2     | 서울 전체 (10,992개) | 0.00001 ✅ | 통계적 신뢰성 확보 | **70%** |
| 성수동 모델 | 성수동만 (104개)     | 0.481 ❌   | 성수동 특성 반영   | **30%** |

### 가중치 설정 근거

- Model 2: 통계적으로 유의 (P < 0.05) → 높은 가중치
- 성수동 모델: 통계적 유의성 없으나, 성수동 고유 특성 추출 목적 → 보정 요소로 활용

## 📊 사용 변수

### 종속변수

- **MZ*유동*증가율**: MZ세대(20~30대) 유동인구의 분기별 증가율

### 독립변수

| 변수명               | 설명                  | 계산식                                |
| -------------------- | --------------------- | ------------------------------------- |
| 집객시설\_수         | 해당 지역 상업시설 수 | -                                     |
| 지하철*역*수         | 지하철역 개수         | -                                     |
| 운영*영업*개월\_평균 | 점포 평균 영업 기간   | -                                     |
| 상권*에너지*지수     | 상권 활성도           | 가중*총매출 × 집객시설*수             |
| 상권*유입*강도       | 외부 유입 정도        | 총*유동인구*수 / (총*상주인구*수 + 1) |
| 주말*매출*비중       | 주말 매출 비율        | 주말*매출 / 전체*매출                 |
| MZ*매출*비중         | MZ세대 매출 비율      | (20대*매출 + 30대*매출) / 전체\_매출  |

## ⚠️ 한계점 및 주의사항

### 1. 상권*유입*강도 과대 반영

- 성수동 모델에서 가중치가 1721로 매우 큼
- 하이브리드 점수에서 이 변수가 결과를 지배할 수 있음
- **개선 방안**: 점수 정규화 후 결합

### 2. 성수동 모델의 통계적 한계

- Prob(F) = 0.481로 통계적으로 유의하지 않음
- 따라서 "예측"이 아닌 "특성 파악"용으로만 활용
- 70:30 비율로 Model 2에 더 높은 가중치 부여

### 3. 정성적 요인 미반영

- SNS 바이럴, 인스타그래머블 장소
- 독특한 카페/음식점
- 임대료 수준
- 이러한 정성적 요인은 별도 분석 필요

### 4. R² 값이 낮음 (0.3%)

- 모델의 설명력은 제한적
- 단, Prob(F) < 0.05로 모델 전체는 유의함
- 탐색적 분석(Exploratory Analysis)으로 활용 권장

## 🔮 향후 연구 방향

1. **점수 정규화**: MinMaxScaler로 두 점수를 0~1로 정규화 후 결합
2. **민감도 분석**: 6:4, 7:3, 8:2 등 다양한 비율 테스트
3. **추가 변수 발굴**: 네이버 검색 언급량, 유튜브 비디오 조회수, 카페 밀집도, 임대료 등
4. **시계열 분석**: 성수동이 "뜨기 전" 데이터로 초기 신호 포착
